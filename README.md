# classifier-java
A port of [classifier](https://github.com/bhrigu123/classifier) to Java

## Description
Organizes your files by extension. Can be used on arbitrary directories. Includes an undo function through [git](https://git-scm.com).
* Audio
* Ringtones
* Videos
* Pictures
* Archives
* Documents
* Webpages
* Programming files
* Plain text
* Books
* Packages
* Programs
